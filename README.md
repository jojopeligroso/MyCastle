# ESL MCP Specification  
This repository contains the model specifications for the ESL web platform, including MCP architecture, resources, tools, and prompts for Admin, Teacher, and Student modules.

## Structure
- **spec/** – detailed specifications and documentation  
- **notes/** – development notes and draft material  

## Purpose
To serve as the authoritative source for defining, testing, and validating all MCP components prior to build phase.
